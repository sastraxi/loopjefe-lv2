### Sample WAV files

Gitignored test wav files for time-stretching and looping.

All files: 30s, 48 kHz, mono, 16-bit PCM WAV. Each is sliced from the middle of a longer source (or the full source if shorter than 60s), resampled with ffmpeg where necessary, and downmixed to mono (L+R average). Intermediate full-length source files (Internet Archive WAV, ccMixter FLAC/MP3) were downloaded into this directory, processed, and then deleted to keep the directory small.
